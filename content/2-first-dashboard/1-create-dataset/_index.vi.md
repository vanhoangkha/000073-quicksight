#### Tải Dataset lên hệ thống QuickSight

1. Ở thanh điều hướng bên trái, chọn **Datasets**. Sau đó, chọn **New dataset**.
![AWS](/images/2/1.2-1.png?width=90pc)

2. Chọn **Upload a file** để tải file có sẵn trong phần [**Giới thiệu**](../../).
![AWS](/images/2/1.2-2.png?width=90pc)

3. Chọn **Next** để xác nhận file đã tải.
![AWS](/images/2/1.2-3.png?width=90pc)

4. Chọn **Visualize** để bắt đầu biểu diễn data.
![AWS](/images/2/1.2-4.png?width=90pc)

5. Chúng ta sẽ được điều hướng tới trang Dashboard mặc định như hình dưới.

![AWS](/images/2/1.2-4a.png?width=90pc)

Tiếp theo, chúng ta sẽ tiến hành thực hiện tạo các biễu diễn (visual) trên QuickSight.
