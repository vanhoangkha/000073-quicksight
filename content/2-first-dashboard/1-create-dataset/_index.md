#### Upload Dataset to QuickSight system

1. In the left navigation bar, select **Datasets**. Then select **New dataset**.
![AWS](/images/2/1.2-1.png?featherlight=false&width=90pc)

2. Select **Upload a file** to download the file available in [**About**](../../).
![AWS](/images/2/1.2-2.png?featherlight=false&width=90pc)

3. Select **Next** to confirm the downloaded file.
![AWS](/images/2/1.2-3.png?featherlight=false&width=90pc)

4. Select **Visualize** to start displaying data.
![AWS](/images/2/1.2-4.png?featherlight=false&width=90pc)

5. We will be navigated to the default Dashboard page as shown below.

![AWS](/images/2/1.2-4a.png?featherlight=false&width=90pc)

Next, we will proceed to create visuals on QuickSight.
