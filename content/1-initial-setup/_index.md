#### Preparation 

In this section, we will create a QuickSight Enterprise account to use for performing data representation.
If you haven't signed up for QuickSight, follow these steps.

#### Sign up for QuickSight
1. Go to [**AWS Console**](https://console.aws.amazon.com), type `QuickSight` in the search bar, and access the service.
![AWS](/images/1/1.1-1.png?featherlight=false&width=90pc)

2. Click **Sign up for QuickSight**.
![AWS](/images/1/1.1-2.png?featherlight=false&width=90pc)

3. We will subscribe to the QuickSight Enterprise version (After the lab is done, you can cancel the subscription). Select **Enterprise** > **Continue**.
![AWS](/images/1/1.1-3.png?featherlight=false&width=90pc)

4. Select **Use IAM federated identities & QuickSight-managed users**.
  - Under **Select a region** , click **Asia Pacific(Singapore)**.
  - Set **QuickSight account name** as you like (eg `admin-created date`).
  - Set email to receive arbitrary notifications.
  - Scroll to the bottom of the page and click **Finish**.
![AWS](/images/1/1.1-4.png?featherlight=false&width=90pc)

5. After successful creation, select **Go to Amazon QuickSight** to be directed to the QuickSight homepage.
![AWS](/images/1/1.1-5.png?featherlight=false&width=90pc)

6. At the Welcome to QuickSight page, click the **X** icon to close the Welcome page.

![AWS](/images/1/1.1-6.png?featherlight=false&width=90pc)

We will proceed to build the QuickSight dashboard in the next step.
