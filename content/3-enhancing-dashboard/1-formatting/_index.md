#### Optional - Keep backup dashboard
1. Select **Save as**. Name the new analysis file (eg `Module 2`), and select **Save**

#### Create custom theme

1. In the left navigation bar, select **Themes** > **Starter themes** > **Seaside**. Select the **...** icon in the right corner of the Seaside theme, and select **Save As**.

![AWS](/images/3/1.3-2a.png?featherlight=false&width=90pc)

2. Give the theme an arbitrary name (eg `Custome Theme`).

3. In the **Font** section, select the arrow > **Inter** to change it to Inter font.

![AWS](/images/3/1.3-2b.png?featherlight=false&width=90pc)

4. In the left navigation bar, select **Data**.

5. To change an existing color, select the color you want to change and select **Custom color**.

![AWS](/images/3/1.3-2c.png?featherlight=false&width=90pc)

6. Change to your desired color (eg `#0EA7AF`). Select **Apply**

![AWS](/images/3/1.3-2d.png?featherlight=false&width=90pc)

7. To add color to the theme, select the **+** icon and select **Custom color**.

![AWS](/images/3/1.3-2e.png?featherlight=false&width=90pc)

8. Add the color you want (eg `CA3C3C`). Select **Apply**.

![AWS](/images/3/1.3-2f.png?featherlight=false&width=90pc)

9. In the left navigation bar, select **Layout** > **Border** to remove the chart border.

10. Select **Save** to save the newly created theme.
![AWS](/images/3/1.3-2g.png?featherlight=false&width=90pc)

11. If the theme has not been applied, open **My themes** > **Custom Theme**.

![AWS](/images/3/1.3-2h.png?featherlight=false&width=90pc)

#### Information field format

1. Select the information field you want to format. In this example, select the **...** icon in the Discount section, select **Show as: Number** > **Percent**. As such, when you use this field, the data will show up as a percentage.

![AWS](/images/3/1.3-3a.png?featherlight=false&width=90pc)

2. Similar to the Sales field: select the **...** icon in the Sales section, select **Format** > **More formatting options**. With this selection, you can change more types of formats.

![AWS](/images/3/1.3-3b.png?featherlight=false&width=90pc)

3. In the Field section, select **Data Type** and change it to **Currency**. Under **Decimal places**, select **0**. As such, when you use this field, the data is displayed as a foreign currency integer.

![AWS](/images/3/1.3-3c.png?featherlight=false&width=90pc)
