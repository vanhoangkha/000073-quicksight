#### Install filters ( Filter )

**Cascading Filter** limits the options displayed in one filter based on the selection made in another filter. For example, filter a list of cities based on a selection in the country filter. Little features like these save users time and make the dashboard highly interactive.

1. In the left navigation bar, select **Filter**. Select the **+** icon to the right of the Filter column. Select **Region**.

![AWS](/images/4/1.4-2a.png?featherlight=false&width=90pc)

2. Select the filter you just created. Select the **Only this visual** section arrow and select **All applicable visuals**.
3. Select the **...** icon in the right corner of the filter and select **Add to sheet**.

![AWS](/images/4/1.4-2b.png?featherlight=false&width=90pc)

4. Repeat the above steps for **Subregion**. In the left navigation bar, select **Filter**. Select the **+** icon to the right of the Filter column. Select **Subregion**.

![AWS](/images/4/1.4-2c.png?featherlight=false&width=90pc)

5. Select the filter you just created. Select the **Only this visual** section arrow and select **All applicable visuals**.
6. Select the **...** icon in the right corner of the filter and select **Add to sheet**.

![AWS](/images/4/1.4-2d.png?featherlight=false&width=90pc)

7. Collapse controls to fit in the dashboard.

![AWS](/images/4/1.4-2e.png?featherlight=false&width=90pc)

8. Select the gear icon in the right corner of the **Region** filter.
9. In the Style section, select **Dropdown** > **Apply**.

![AWS](/images/4/1.4-2f.png?featherlight=false&width=90pc)

10. Select the gear icon in the right corner of the **Subregion** filter.
11. Select **Show relevant visuals only**.

![AWS](/images/4/1.4-2g.png?featherlight=false&width=90pc)

12. Select **Region equals (Drop down)** > **Update**.
13. Select **Apply**.

![AWS](/images/4/1.4-2h.png?featherlight=false&width=90pc)
